IPython notebook for microbial ecology analysis in QIIME

What is included in this download?

There are 6 files included in the download. 1) the IPython notebook, 2) a test sequence file in fasta format, 3) a test QIIME mapping (metadata) file, 4) a parameters file, 5) an ugly perl script that is used in the workflow, and 6) a README file with some notes.
